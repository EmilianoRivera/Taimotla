# Taimotla
Repositorio para Aplicación web destinada a la Fundación Futuro con Derechos enfocado a niños, niñas y adolescentes

1. Crear un entorno virtual
python -m venv .venv
# En Windows:
.venv\Scripts\activate

2. Instalación de Librerias y Flask
pip install flask psycopg2-binary

3. Instala pgAdmin y crear la bd
En el proyecto para hacer la conexión con la bd modifica el archivo database.py y pon tus credenciales.
